# ♟ Clace

> A chess engine written in C/C++, built for fun.

---

## Overview

Clace is a hobby chess engine featuring two search implementations, basic UCI protocol support, and a lightweight graphical interface.

## Features

- **NegaMax** and **Alpha-Beta** search engines
- Basic **UCI** command handling
- Minimal **GUI** built with [Raylib](https://www.raylib.com/)

## Status

Everything is a work in progress. Expect rough edges, experiments, and occasional chaos.
